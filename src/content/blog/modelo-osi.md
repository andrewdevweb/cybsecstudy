## ¿Qué es el Modelo OSI?

El Modelo OSI es un framework de 7 capas que define cómo los 
dispositivos en red envían, reciben e interpretan datos.

## Las 7 Capas

| Capa | Nombre | PDU |
|------|--------|-----|
| 7 | Aplicación | Datos |
| 6 | Presentación | Datos |
| 5 | Sesión | Datos |
| 4 | Transporte | Segmento |
| 3 | Red | Paquete |
| 2 | Enlace de Datos | Trama |
| 1 | Física | Bit |
